# Vector Operations in Rust

This repository is a personal learning project dedicated to exploring and practicing the use of **Vectors** in the Rust programming language.

## About the Project
The main goal of this project is to deepen my understanding of how to work with dynamic arrays (vectors), implement basic mathematical operations, and manage input/output processes in Rust. 

This is a practice-oriented project, focusing on:
* Learning how to manipulate `Vec<T>` data.
* Implementing functions for basic statistics (sum, max, min, average).
* Handling user input and error checking using `match` and `loop`.
* Structuring code with modules.

## Features
- **Vector Summation**: Calculates the total sum of vector elements.
- **Min/Max Search**: Identifies the smallest and largest values in a vector.
- **Average Calculation**: Computes the arithmetic mean of the elements.
- **Dynamic Input**: Allows users to define the number of elements and populate the vector through the command line.

## Learning Progress
This project is part of my journey to learn Rust. It helps me improve my understanding of:
- Memory management and ownership.
- Effective use of Rust standard library features.
